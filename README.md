# Калькулятор на Python

Простой консольный калькулятор с поддержкой основных арифметических операций.  
Это мой первый проект, который я выложил на GitHub.

## 🛠️ Функционал
- Сложение (+)
- Вычитание (-)
- Умножение (*)
- Деление (/) с проверкой на ноль
- Возведение в степень (**)
- Проверка чисел на чётность

## 🚀 Как запустить
1. Скачай файл calculator.py.
2. Открой терминал в папке с файлом.
3. Выполни команду:  
   python calculator.py

## 🧑‍💻 Используемые технологии
- Python 3
- Модули: random, time
